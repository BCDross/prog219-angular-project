Kanban Web Tracker

Overview

This web app is a Kanban or card-based system task tracker. This allows you to quickly see progress on tasks of an ongoing project. As well as maintain more granular data of the

Tasks

The Tasks link above will take you to your tasks list. There you will be able to add, update, and archive or delete the tasks.

Tasks by status

The Tasks by status link above will take you to a page with your tasks organized by status. This is a quick view that can be presented to managers or others as a quick update page.
